qenru Form Validation

How to use

1st Step >> Link required files

<!-- jQuery library (served from Google) -->
                                <script src="//ajax.googleapis.com/ajax/libs/jquery/1.8.2/jquery.min.js"></script>
                                <!-- qenru Form Validation Javascript file -->
                                <script src="/js/jquery.qenruformvalidation.min.js"></script>
                                <!-- Sweet Alert Library (served from Sweet Alert) -->
                                <script src="/js/sweetalert2.js"></script>
                                <!-- jQuery Masked Input Library (served from digitalbush.com) -->
                                <script src="/js/jquery.maskedinput.min.js"></script>
                                <!-- qenru Form Validation CSS file -->
                                <link href="/css/jquery.qenruformvalidation.css" rel="stylesheet" >

2nd Step >> Link optional files

<!-- Bootstrap library (served from Bootstrap) -->
                                    <script src="/js/bootstrap.min.js"></script>
                                    <!-- Bootstrap CSS file -->
                                    <link href="/css/bootstrap.css" rel="stylesheet" >
                                    <!-- Fontawesome CSS file -->
                                    <link href="/css/font-awesome.min.css" rel="stylesheet" >

3rd Step >> HTML markup

<!-- Input Text ( Required Format )-->
                                        <input type="text" class="input-group" data-required="true">
                                        <span class="form-error"></span>

                                        <input type="radio" class="input-group" data-required="true">
                                        <span class="form-error"></span>

                                        <input type="checkbox" class="input-group" data-required="true">
                                        <span class="form-error"></span>

                                        <!-- Input Password ( minimum character limitation )-->
                                        <!-- data-label ( Define field name for error message with data-label attribute )-->
                                        <input type="password" class="input-group" data-min="6" data-label="Password">
                                        <span class="form-error"></span>

                                        <!-- Input Password ( maximum character limitation )-->
                                        <!-- data-type ( Define data-type as text to validate elements like type text )-->
                                        <textarea data-type="text" class="input-group" data-max="30" data-label="Message">
                                        <span class="form-error"></span>

                                        <!-- Input Password & Password Again ( Value matching validation )-->
                                        <!-- data-rel ( Check fields which has the same data-rel value )-->
                                        <input type="password" class="input-group" data-match="true" data-rel="1">
                                        <span class="form-error"></span>

                                        <input type="password" class="input-group" data-match="true" data-rel="1">
                                        <span class="form-error"></span>

                                        <!-- Input Email Validation -->
                                        <input type="text" class="input-group" data-type="email">
                                        <span class="form-error"></span>

                                        <!-- Input Phone Validation -->
                                        <input type="text" class="input-group" data-type="tel">
                                        <span class="form-error"></span>

4th Step >> Call the qenru Validation

$("#formID").qValidate();
  • VERSION 1.0.0 includes features of >>
  • Required fields validation
  • Values of fields matching validation
  • Checkbox & Radio input validation
  • Min & Max character length validation
  • Alert style option ( inline , sweet alert )
  • Input mask options
  • Input file validation
  • Ajax form request
  • Ajax method & other attributes management
  • Ajax response management ( via functions )
